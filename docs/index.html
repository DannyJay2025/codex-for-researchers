低价科研工作流，先把论文和项目整理清楚。

面向学生、科研作者和小团队：用 Codex 工作流完成论文精读、引用审计、复现包审计。付款后通过微信交付 Markdown 结果。

1.99 元 论文精读：发送论文链接、摘要或片段，收到结构化阅读笔记。
3.99 元 引用审计：发送段落或 claim，收到 claim-to-citation 风险表。
5.99 元 复现包审计：发送仓库或文件清单，收到复现检查表。

怎么下单

1. 加微信

发送你要购买的服务：论文精读、引用审计或复现包审计。

2. 付款发材料

扫码付款后发送截图，以及论文、段落、claim 或项目说明。

3. 收到交付

收到 Markdown 结果，可用于组会、写作准备或投稿前检查。

你会收到什么

论文精读

一句话主旨、方法地图、关键证据、局限和后续问题。

引用审计

原子 claim 拆分、证据需求、风险标记和更稳妥写法。

复现包审计

README、环境、代码入口、数据、图表和可用性声明检查。
