# 客户下单说明

这是一个低价科研工作流服务入口，适合先快速试用。

## 当前价格

| 服务 | 价格 | 你需要发送 | 你会收到 |
| --- | --- | --- | --- |
| 论文精读 | 1.99 元 | 论文链接、摘要、PDF 摘录或正文片段 | Markdown 论文精读笔记 |
| 引用审计 | 3.99 元 | 一段论文文字或 claim 列表 | claim-to-citation 风险表 |
| 复现包审计 | 5.99 元 | GitHub 链接、文件清单或项目说明 | 复现检查表和风险建议 |

## 下单流程

1. 加微信：`zhiyanaishe`
2. 说明你要购买哪一项服务。
3. 扫码付款。
4. 发送付款截图和材料。
5. 等待 Markdown 交付结果。

## 建议发送格式

```text
服务：论文精读 / 引用审计 / 复现包审计
语言：中文 / 英文 / 中英双语
材料：论文链接、摘要、段落或项目说明
用途：组会 / 写论文 / 投稿前检查 / 复现整理
是否允许脱敏后作为公开案例：是 / 否
```

## 交付边界

- 不保证论文录用。
- 不保证引用一定正确，最终投稿前需要你自己核对。
- 不提供医学、法律、财务等专业意见。
- 不会公开你的私人论文、付款截图或未发表数据。

## 常见问题

下单前如果不确定是否适合，请先看 `faq.zh-CN.md`。里面说明了服务范围、退款边界、隐私材料、是否代写、是否保证引用正确等问题。
