# 复购升级手册

这份手册用于把 1.99/3.99/5.99 元低价引流单自然升级为小包、完整审计或定制服务。原则是：先交付有用结果，再轻量推荐下一步，不强卖，不扩大承诺。

## 升级梯度

公开报价梯度在：

```text
ops/offer-ladder.csv
```

核心路径：

| 首单 | 推荐下一单 | 价格 | 适合场景 |
| --- | --- | ---: | --- |
| 1.99 论文精读 | 10 篇论文快速精读包 | 19.9 元 | 组会、开题、文献扫描 |
| 3.99 引用审计 | 一节 manuscript 引用审计 | 39.9 元 | 投稿前检查、返修前自查 |
| 5.99 复现包审计 | 完整复现准备度审计 | 99 元 | 代码/数据发布、投稿材料整理 |
| 任意服务 | 小型实验室工作流设置 | 699 元起 | 固定写作风格、模板和 SOP |

低价首单只用于建立信任，不要在客户还没确认价值前推高价。

## 自动生成推荐

交付后运行：

```powershell
python scripts\suggest_offer.py --after-service paper-reading
```

可选：

```powershell
python scripts\suggest_offer.py --after-service citation-audit
python scripts\suggest_offer.py --after-service repro-audit
```

## 推荐时机

只在这些情况推荐：

- 客户回复“有帮助”。
- 客户问还有没有更多论文/段落/项目可以处理。
- 客户提出连续需求。
- 客户要求完整稿件、完整 repo 或团队流程。

不要在这些情况推荐：

- 首单还没交付。
- 客户对结果不满意。
- 客户材料明显超出你能处理的边界。
- 客户要求代写、伪造引用、保证发表。

## 交付后话术

论文精读后：

```text
如果这次结果对你有帮助，后面可以做 10 篇论文快速精读包，19.9 元。适合组会、开题或先搭一个文献地图。不用现在决定，确认这次交付有用后再说。
```

引用审计后：

```text
如果这次 claim 检查对你有帮助，下一步可以做一节 manuscript 引用审计，39.9 元。范围是一节正文，不做系统综述，也不承诺最终引用一定正确，但会帮你把高风险表述和需要核对的位置标出来。
```

复现包审计后：

```text
如果你准备正式整理代码和数据发布，可以做完整复现准备度审计，99 元。范围是一个项目包，交付 repo 清单、阻塞点、README/环境/数据建议和可用性声明草稿。
```

实验室或团队：

```text
如果你们团队后面会反复做论文精读、引用审计或复现整理，可以单独做一个小型实验室工作流设置，699 元起。会交付私有模板、SOP 和学生可复用的 Codex 提示流程。
```

## 范围边界

升级服务仍然不包含：

- 代写论文。
- 伪造引用、数据或审稿意见。
- 保证录用。
- 医学、法律、财务或伦理合规意见。
- 未脱敏敏感材料的公开处理。

## 记录复购

每次推荐后，在私有表里记录：

```text
orders/growth-metrics.csv
orders/order-tracker.csv
```

建议记录：

- 推荐了哪个 offer。
- 客户是否有兴趣。
- 是否付款。
- 为什么拒绝。
- 是否需要补 FAQ、样例或文案。
