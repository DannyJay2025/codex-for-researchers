# 常见问题

这份 FAQ 面向准备下单的用户，帮助你在付款前判断服务是否适合当前需求。

## 这是不是代写论文？

不是。这个项目只做科研阅读、结构化整理、引用风险提示和复现材料检查，不代写论文，不伪造数据，不编造引用，也不承诺发表或录用。

## 1.99 元论文精读包含什么？

适合一篇论文的快速理解。你可以发送论文链接、摘要、引言、方法片段或 PDF 摘录。交付通常包括：

- 论文核心问题和一句话结论。
- 方法和证据链梳理。
- 主要局限和阅读风险。
- 适合组会或继续阅读的追问。

如果只提供摘要或片段，结果会标注为“有限范围阅读”，不会假装已经读完整篇。

## 3.99 元引用审计包含什么？

适合检查一段论文文字或少量 claims 是否需要引用支撑。交付通常包括：

- 将文本拆成可检查的 claims。
- 标出强支撑、弱支撑、缺支撑和过度表述风险。
- 给出更稳妥的改写建议。
- 给出下一步应检索或核对的方向。

低价单不包含完整系统综述，也不保证找到最终可投稿引用。

## 5.99 元复现包审计包含什么？

适合在复现前先看项目是否容易卡住。你可以发送 GitHub 链接、README、文件清单或项目说明。交付通常包括：

- README、环境、入口脚本、数据、输出和许可证检查。
- 复现阻塞点和缺失材料提示。
- 代码/数据可用性声明草稿建议。
- 下一步整理优先级。

低价单不包含完整跑通实验、修复代码或重做结果。

## 多久交付？

当前是个人手动交付，默认 24-72 小时。低价单不承诺实时返回。如果材料范围过大，会先要求缩小范围或单独报价。

## 付款后需要发什么？

请发送：

```text
服务：论文精读 / 引用审计 / 复现包审计
语言：中文 / 英文 / 中英双语
材料：论文链接、摘要、段落、claim 列表或项目说明
用途：组会 / 写作 / 投稿前检查 / 复现整理
是否允许脱敏后作为公开案例：是 / 否
付款截图：已发送
```

## 可以发未发表论文或隐私数据吗？

不建议直接发送敏感材料。请尽量先脱敏，或只发送摘要、片段、目录结构和非敏感说明。付款截图、客户材料、未发表论文、私有数据和交付结果不会提交到公开仓库。

## 会保证引用一定正确吗？

不会。引用审计会指出风险和需要核对的位置，但最终投稿前仍需要你自己或合作者核对原文、DOI、页码、统计结果和期刊要求。

## 可以退款吗？

如果尚未开始处理，且材料明显不适合当前服务，可以协商退款或换材料。若已经开始处理，但客户临时大幅改变范围，当前单按原范围交付，新范围需要重新下单。

如果需求涉及代写、伪造引用、保证发表、处理未脱敏敏感隐私数据等不适合内容，会拒绝并视情况退款。

## 最重要的边界是什么？

这是 AI 辅助科研工作流草稿，不替代导师、同行评审、专业编辑、统计顾问、法律/医学/伦理/合规意见。所有引用、事实、数据和投稿要求都需要最终人工复核。
