## What To Put In The Private Version

Each premium pack should include:

- A Codex skill or skill extension.
- A request template.
- A finished output template.
- A quality checklist.
- One realistic redacted example.
- A short sales note explaining the result users get.

## Public Teaser Rule

Expose enough to show value:

- Pack name.
- Use case.
- Screenshot or short excerpt.
- Buyer outcome.

Do not expose:

- Full paid prompts.
- Complete checklists.
- Private examples.
- Client-specific SOPs.
