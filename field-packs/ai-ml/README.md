# AI/ML Field Pack

## Target Users

- Machine learning researchers
- Applied AI teams
- Technical founders preparing white papers
- Students writing conference submissions

## Paid Workflows

- Benchmark claim audit.
- Ablation and baseline checklist.
- Reproducibility package review.
- Model card and dataset card drafting.

## Field-Specific Risks

- Claiming state of the art without verified benchmark comparison.
- Omitting compute, seed, hyperparameter, or dataset split details.
- Comparing against weak or outdated baselines.
- Reporting improvements without confidence intervals or significance checks when needed.

## Premium Assets To Build

- Benchmark evidence table.
- Reproducibility checklist for ML repositories.
- Model card template.
- Reviewer-risk checklist for conference submissions.
