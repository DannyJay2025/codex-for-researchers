# Sample Prompts

## Paper Reading

```text
Use research-paper-reader on this PDF. Create a journal club reading brief with the paper logic, figure walkthrough, limitations, and three discussion questions.
```

## Citation Support

```text
Use citation-support-finder on the introduction below. Split it into atomic claims, flag unsupported claims, and produce a claim-to-citation map.
```

## Manuscript Polishing

```text
Use manuscript-polisher in journal-ready mode. Preserve all scientific claims and return revised text plus risk notes.
```

## Reproducibility Packaging

```text
Use reproducibility-packager to audit this project folder. Draft code and data availability statements, but do not claim public release unless the files prove it.
```
